üÜò TROUBLESHOOTING

    Q: Getting 422 error?
    A: Restart the server. API expects JSON format.

    Q: No suppliers showing?
    A: This is expected! Import suppliers first via CSV.

    Q: Login not working?
    A: Check browser console for errors. Clear cache.

    Q: Chatbot not responding?
    A: Chatbot is basic NLP. Try: "Find suppliers in Texas"

    Q: Filters not working?
    A: Import suppliers first, then use filters.
