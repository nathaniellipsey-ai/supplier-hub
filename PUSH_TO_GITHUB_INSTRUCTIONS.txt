                 TROUBLESHOOTING
=================================================================

Problem: "fatal: not a git repository"
Solution: Make sure you're in the right directory
  cd "C:\Users\n0l08i7\Desktop\New folder\supplier-hub"

Problem: "Permission denied" or "403 Forbidden"
Solution: Ensure you have push access and valid GitHub credentials
  git config --global user.name "Your Name"
  git config --global user.email "your@email.com"

Problem: "Please tell me who you are" error
Solution: Configure Git user first
  git config user.name "Your Name"
  git config user.email "your@email.com"
  
Problem: "Everything up-to-date" but files show as changed
Solution: Files may not be staged. Run:
  git add -A
  git status (to verify)
  git commit -m "..."

Problem: Can't push to main branch
Solution: Check if main is the right branch
  git branch (shows current branch)
  git push origin [your-branch-name]
