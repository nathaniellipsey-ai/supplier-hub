OPTION 1: RUN LOCALLY (TESTING)

Quick Start:
  1. Double-click: START_DASHBOARD.bat
  OR manually run:
     cd supplier-hub

  2. Open browser: http://localhost:8000

  3. You should see:
     ‚úì Dashboard loads (or login page)
     ‚úì All pages work

OPTION 2: DEPLOY TO RENDER (PRODUCTION)
----------------------------------------

Step 1: Prepare Git
  git init
  git add .
  git commit -m "Deploy: Supplier Hub Dashboard"
  git remote add origin https://github.com/YOUR_USERNAME/supplier-hub.git
  git push -u origin main

Step 2: Create Render Service
  1. Go to https://render.com
  2. Sign up/login
  3. Click "New +" ‚Üí "Web Service"
  4. Connect GitHub repository
  5. Configure:
     - Name: supplier-hub
     - Environment: Python 3
     - Build Command: pip install -r requirements.txt
     - Start Command: uvicorn app:app --host 0.0.0.0 --port 8000

Step 3: Deploy
  - Click "Create Web Service"
  - Wait 2-3 minutes
  - Your dashboard lives at: https://supplier-hub.onrender.com


OPTION 3: DEPLOY TO HEROKU
--------------------------

  heroku login
  heroku create supplier-hub
  git push heroku main
  heroku open
