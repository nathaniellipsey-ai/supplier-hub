üìã CSV Format

Your CSV file must contain these columns:

Required Columns:

  • id - Unique supplier ID (number)
  • name - Supplier name
  • category - Product category
  • location - City, State (e.g., "New York, NY")
  • region - State code (e.g., "NY")
  • rating - Rating 0-5 (decimal)
  • aiScore - AI Score 0-100 (number)
  • products - Products separated by ; (e.g., "Product A;Product B")
  • certifications - Certifications separated by ; (e.g., "ISO 9001;EPA")
  • walmartVerified - true or false
  • yearsInBusiness - Years in business (number)
  • projectsCompleted - Number of projects (number)
